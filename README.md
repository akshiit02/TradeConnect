# TradeConnect

A responsive marketing site for an export-sourcing brokerage that connects international buyers with verified Indian suppliers of moringa powder and other agricultural products.

🌐 **[View live site](https://akshiit02.github.io/TradeConnect/)**

## What it does

TradeConnect positions the business as a sourcing intermediary, not a manufacturer — it vets 2–4 matching suppliers per inquiry and manages the buyer-seller relationship end to end. The site walks through what the service does, the products it sources, how the process works, and ends with a working contact form for enquiries.

## Features

- Fully responsive layout with a mobile nav toggle
- Scroll-based reveal animations
- Active-section nav highlighting as you scroll
- Contact form wired to a real backend (Web3Forms) — submissions actually deliver to an inbox, not a dead form

## Built with

- HTML5, CSS3 (hand-written, no framework)
- Vanilla JavaScript

## Run it locally

```bash
git clone https://github.com/akshiit02/TradeConnect.git
cd TradeConnect
```
Open `index.html` in a browser — no build step needed.
